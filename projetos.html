Calculadora-IMC

Como meu primeiro projeto em desenvolvimento web, esta calculadora de IMC foi fundamental para iniciar minha jornada com HTML, CSS e JavaScript. Através dela, pude ter minha primeira experiência prática com a lógica de programação e a criação de interfaces, aprendendo a estruturar, estilizar e dar funcionalidade a uma aplicação. O projeto, que calcula e classifica o IMC com base nos padrões da Organização Mundial de Saúde (OMS), solidificou minha base para futuros aprendizados.
